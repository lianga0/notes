# Micron HBME3 – lower power, greater capacity

Micron's industry-leading HBM3E 36GB 12-high delivers significantly lower power consumption than our competitors’ 8-high 24GB offerings, despite having 50% more DRAM capacity in the package.

Micron HBM3E 12-high boasts an impressive 36GB capacity, allowing larger AI models like Llama 2 with 70 billion parameters to run on a single processor. This capacity increase allows faster time to insight by avoiding CPU offload and GPU-GPU communication delays.

Micron HBM3E 12-high 36GB offers more than 1.2 terabytes per second (TB/s) of memory bandwidth at a pin speed greater than 9.2 gigabits per second (Gb/s).

`pin speed` 指的是“每个数据引脚（I/O线）的有效数据传输速率”，也就是每条数据线每秒能传多少比特（Gb/s）。不是指某个“针脚的机械速度”，而是该电信号通道的比特率。“Gb/s per pin”已经包含了 DDR 的传输特性（即每个时钟的双边沿传输），所以这是有效数据率，而不是单纯的时钟频率。
